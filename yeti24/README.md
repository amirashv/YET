# YETI24

Reproducible build & run pipeline for the YETI-24 prototype:
- Dual eye-camera tracking
- IMU/sensor logging pipeline
- Calibration and example outputs

See: docs/quickstart.md (to be added)




## CAD Files

Mechanical components for the YETI24 prototype are located in:

- cad/step/  → engineering-grade STEP files
- cad/stl/   → printable STL files

All parts were designed in Onshape.
